
## Where to work — Forgejo-first

- **Canonical origin:** Forgejo `git@git.langevc.com:capacium/claude-code-capacium.git` — work **here**.
- **GitHub** `github.com/Capacium/claude-code-capacium` is a read-only mirror (force-pushed from Forgejo) — do **not** push there.
- **Local clone:** `~/Documents/repositories/forgejo/capacium/claude-code-capacium` (layout `<provider>/<org>/<repo>`); remotes `origin`=Forgejo, `github`=mirror.
- Pull requests on Forgejo. CI: Forgejo for dev; some workflows are guarded to GitHub (hybrid) — see `capacium-internal-docs/docs/develop/forgejo-first-workflow.md`.
